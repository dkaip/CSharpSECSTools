Class BodyOutputConfig

The properties of this class control the output format of SECSItem objects or in other words the body or content of a SECS message.

DisplayAsType

This property controls how a SECSItem is displayed in the case where a formatter is configured to output using XML.

Declaration
public DisplayAsType DisplayAsType { get; set; }
Property Value
Type Description
DisplayAsType

If this property's value is DisplayAsType.Elements a message's body will be displayed using XML elements for the various SECSItem(s) elements therein. If the value is DisplayAsType.Attributes the body will be displayed using XML attributes for the various SECSItem(s) elements therein.
Remarks

This property's value has no effect if the output ends up being SML.

The default is to display as Attributes.

DisplayCount

If the display format is SML this property controls whether or not "count" values are displayed in the output. These "count" values indicate how many elements are in the individual SECSItems.

Remarks

This property only has an effect on output if the display format is SML.

The default is true.
