Namespace com.CIMthetics.CSharpSECSTools.TextFormatter

Classes

BodyOutputConfig

The properties of this class control the output format of SECSItem objects or in other words the body or content of a SECS message.

HeaderOutputConfig

The properties of this class control the output format of SECSHeader objects.

SECSFormatter

This is an abstract base class providing the framework for classes that will be able to turn SECSMessages, SECSHeaders, and SECSITEMs into a C# string (or append the results to a supplied StringBuilder) producing output suitable for output to a terminal and or a file.

Note: At this time the current max indentation level is 132 spaces.

SECSFormatterFactory

A factory class used to create a SECSFormatter of the appropriate type.

SMLFormatter

A class to create a textual representation in SML of SECSMessages, SECSHeaders, and SECSItems.

SMLOutputConfig

This class is effectively a copy of the class XMLOutputConfig. Its only purpose is to facilitate the "automatic" reading of the TextFormatterConfig section of a appsettings.json file. Refer to the documentation for the class XMLOutputConfig for more information.

TextFormatterConfig

This class is used as the argument of the CreateFormatter method of the SECSFormatterFactory which creates a SECSFormatter object that is implemented to produce logging output as either SML or XML formatted text.

XMLFormatter

A class to create a textual representation in XML of SECSMessages, SECSHeaders, and SECSItems.

XMLOutputConfig

This class contains the information for a formatter to use when producing output.
