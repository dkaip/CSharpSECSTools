DisplayAsType

This property controls how a SECSHeader is displayed in the case where a formatter is configured to output using XML.

Declaration
public DisplayAsType DisplayAsType { get; set; }
Property Value
Type Description
DisplayAsType

If this property's value is DisplayAsType.Elements the message's header will be displayed using XML elements for the various members of the SECSHeader. If the value is DisplayAsType.Attributes the header will be display using XML attributes for the various members of the SECSHeader.
Remarks

This property's value has no effect if the output ends up being SML.

The default is to display as Attributes.

DisplayMessageIdAsSxFy

This property controls how a SECSMessage's stream and function are displayed when the output format is XML.

Declaration
public bool DisplayMessageIdAsSxFy { get; set; }
Property Value
Type Description
System.Boolean

If true, a message's stream and function are output in a form similar to "SxFy". For example, S1F13. If false the message's stream and function are output separately in either attribute or element form.
Remarks

This property's value has no effect if the output ends up being SML.

The default is false.
