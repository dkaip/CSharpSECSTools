Class TextFormatterConfig

This class is used as the argument of the CreateFormatter method of the SECSFormatterFactory which creates a SECSFormatter object that is implemented to produce logging output as either SML or XML formatted text.

MaxIndentionSpaces

This sets the maximum number of spaces available to be used to provide indentation for output text generation.

Declaration
public int MaxIndentionSpaces { get; set; }
Property Value
Type Description
System.Int32

The maximum number of spaces available for indentation.
Remarks

This may sound stupid, but, how can you run out of spaces? Working with strings tends to produce a lot of garbage that the garbage collector will need to deal with. Since many of the programs that use this API may be executing for a long time and generating a lot of output the spaces required for indentation have been preallocated in a string[].

Unlike most other properties this property is only used during the construction of a formatter. Once created this property is not used again. Thus, changing its value will not have any effect.

The default value is 48. Specifying an overly large value for this property does nothing but use more memory. If the value is too small a System.IndexOutOfRangeException will be thrown at the point the CurrentIndentLevel exceeds the value of this property. See SECSFormatter for more information.
