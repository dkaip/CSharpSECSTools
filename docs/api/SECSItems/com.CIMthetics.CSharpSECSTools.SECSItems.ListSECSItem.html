Class ListSECSItem

This class represents a SECSItem with the SECS data type of L, which is a list of SECSItems. From the C# side this data type is handled as a C# LinkedList<SECSItem>.

GetElementAt(String)

This method returns a SECSItem contained in this list based on its "address" or null if the item does not exist.

In the example below a specified address of "3.2" would return the element with the value 'Answer'.

  1. A 'ABC'
  2. A 'DEF'
  3. L 3
    1. I2 -32768
    2. A 'Answer'
    3. U1 255
  4. F4 3.141592
