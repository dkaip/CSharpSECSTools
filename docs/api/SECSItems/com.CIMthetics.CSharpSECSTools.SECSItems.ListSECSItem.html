Class ListSECSItem

This class represents a SECSItem with the SECS data type of L, which is a list of SECSItems.

Declaration
public int Count { get; }
Property Value
Type Description
System.Int32
Remarks

This property only counts the elements in this list. It does not account for any elements within any ListSECSItems that may be contained in this list. For example, if this ListSECSItem has 5 elements that are themselves ListSECSItems each with 5 elements. It will still only return a value of 5.

This returns the same value as the LengthInBytes property.

System.Collections.Generic.Dictionary<System.String, SECSItem>

A Dictionary<string, SECSItem> where the individual elements contained within are accessible via their "address".
Remarks

The keys of the dictionary are the "addresses" of the individual elements. The format of the "address" keys is the same as that used by the GetElementAt(String) method. Refer to it for more information.

It you have a large message that you need to grab a bunch elements from this might be more efficient than calling GetElementAt(String) a bunch of times.

  3. L 4
