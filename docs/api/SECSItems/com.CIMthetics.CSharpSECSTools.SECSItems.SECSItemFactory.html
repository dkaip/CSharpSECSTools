Class SECSItemFactory

This is a static "factory" class that is used to create a SECSItem object from a byte[] that is in "wire/transmission" form. As you might expect the bytes contained within the byte[] must actually be in the form of those of an item in a SECS-II message.

GenerateSECSItem(Byte[], Int32)

Generates a SECSItem object from the byte[] that is passed to it. The parameter offset specifies an offset into the supplied byte[] where the data for the SECSItem starts. The data in the byte[] needs to be in "wire/transmission" format.

Remarks

This would be the desired method to use in the case where the byte[] also contains the header for a SECs message (located at the beginning of the byte[]). In this situation, since a SECS message header is 10 bytes in length you would pass 10 in as the value of the offset parameter. This will cause this method to skip the first 10 bytes of the byte[] and extract / generate the SECSItem from the remaining bytes.
